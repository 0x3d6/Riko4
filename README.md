# Riko4 [![Build Status](https://travis-ci.org/incinirate/Riko4.svg?branch=master)](https://travis-ci.org/incinirate/Riko4)

A fantasy console / game engine

Build Instructions:

```
git clone https://github.com/incinirate/riko4
cd riko4
mkdir build
cd build
cmake ..
make
```
